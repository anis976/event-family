# Internationalisation (FR / EN)

## Règle d’or

**Tout texte visible par l’utilisateur** (interface, flash, e-mail, validation, admin) doit exister en **français et en anglais** dès l’ajout de la fonctionnalité — pas dans un second temps.

## Fichiers

| Fichier | Domaine | Usage |
|---------|---------|--------|
| `translations/messages.fr.yaml` / `.en.yaml` | `messages` | UI (`ui.*`), flashes (`flash.*`), e-mails (`email.*`), admin (`admin.*`), notices (`notice.*`) |
| `translations/validators.fr.yaml` / `.en.yaml` | `validators` | Messages `#[Assert\…]`, `UniqueEntity`, contraintes custom |
| `translations/EasyAdminBundle.fr.yaml` / `.en.yaml` | `EasyAdminBundle` | Boutons, modales, pagination admin |

## Clés à utiliser

- **Templates Twig** : `{{ 'ui.module.key'|trans }}` ou `{% trans %}…{% endtrans %}`
- **Formulaires PHP** : injecter `TranslatorInterface`, préfixer les labels / `constraints` avec `$t('…')`
- **Flashes** : `addSuccessFlash('flash.module.action')` — jamais de phrase en dur
- **Exceptions métier** : `throw new \DomainException('flash.module.reason')`
- **Entités** : `message: 'group.name.required'` (domaine `validators`)
- **E-mails utilisateur** : `TransactionalEmailHelper` + locale du compte
- **E-mails admin** : `prepareForAdmin()` / `transForAdmin()` (toujours FR)
- **Dates Twig** : filtre `|ef_datetime` ou `|ef_date` (pas `|date('d/m/Y')`)
- **Pagination** : `{% include 'components/_ef_paginate.html.twig' %}` + `App\Util\Pagination::create()` ; styles `.ef-paginate`
- **JavaScript client** : passer les textes traduits via attributs `data-ef-alert-*` sur l’élément racine (ex. `.ef-profile-avatar-manager`, formulaire contact) — pas de chaîne en dur dans `assets/js/`

## Admin EasyAdmin

- CRUD : `admin.crud.{entity}.*` via `AbstractAdminCrudController::t()`
- Libellés natifs EA : `EasyAdminBundle.*`

## Checklist avant merge

- [ ] Clés ajoutées dans **messages.fr.yaml** et **messages.en.yaml** (ou validators.*)
- [ ] Aucune chaîne française en dur dans contrôleurs / services / templates
- [ ] Test rapide avec bascule locale (sidebar ou sélecteur admin)
